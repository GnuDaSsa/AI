# Novel Lab

초보자가 혼자 완성 가능한 짧은 비주얼 노벨을 만드는 작업 폴더다.

## 현재 목표

- 엔진: Ren'Py
- 범위: 10~15분 분량 단편
- 구조: 도입 1개, 선택지 2회, 엔딩 3개
- 첫 목표: 플레이 가능한 텍스트 프로토타입 완성

## 폴더 구조

- `docs/`: 기획, 설계, 점검 문서
- `renpy/game/`: Ren'Py 스크립트와 리소스 위치
- `assets/`: 원본 리소스 초안 보관

## 바로 할 일

1. `docs/01-plan/features/first-visual-novel.plan.md` 읽기
2. `docs/02-design/features/first-visual-novel.design.md` 읽기
3. `renpy/game/script.rpy`에서 대사와 선택지 수정
4. 이미지와 음악이 생기면 `renpy/game/images`, `renpy/game/audio`에 추가

## Ren'Py에서 연결할 기본 파일

- 스크립트: `renpy/game/script.rpy`
- 이미지 폴더: `renpy/game/images`
- 오디오 폴더: `renpy/game/audio`
