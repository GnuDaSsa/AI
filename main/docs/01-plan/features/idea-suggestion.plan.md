# Plan: 아이디어 제안소 (Idea Suggestion Box)

**Feature ID**: idea-suggestion
**Created**: 2026-02-27
**Phase**: Plan
**Status**: Draft

---

## 1. 개요 (Overview)

### 배경
Community 섹션에 아이디어 제안소 기능을 추가한다.
AI 클럽 멤버 누구나 간단하게 아이디어를 제안할 수 있는 공간이며,
placeholder(흐린 글씨 예시)가 있는 입력창을 클릭하면 예시가 사라지고 직접 입력할 수 있는 직관적인 UX를 제공한다.

### 목적
- 멤버들이 자유롭게 아이디어를 남길 수 있는 채널 마련
- 로그인 없이 누구나 제안 가능 (접근 장벽 최소화)
- 관리자(Admin)는 제안된 아이디어를 검토하고 상태를 관리

---

## 2. 요구사항 (Requirements)

### 기능 요구사항 (Functional)

| ID | 요구사항 | 우선순위 |
|----|----------|--------|
| FR-01 | 누구나(비인증) 아이디어 제안 가능 | High |
| FR-02 | 제목(선택) + 내용(필수) 입력 폼 | High |
| FR-03 | placeholder로 작성 예시 표시, 클릭 시 사라짐 | High |
| FR-04 | 제출 후 MongoDB에 저장 | High |
| FR-05 | 관리자는 전체 아이디어 목록 조회 가능 | High |
| FR-06 | 관리자는 아이디어 상태 변경 (제안됨 → 검토중 → 채택/보류) | Medium |
| FR-07 | 관리자는 아이디어 삭제 가능 | Medium |
| FR-08 | 일반 사용자에게는 최근 채택된 아이디어 목록 공개 (선택적) | Low |

### 비기능 요구사항 (Non-Functional)

- 기존 page8.py (AI 꿀팁 공유) 패턴과 일관성 유지
- MongoDB 연결 실패 시 친절한 오류 안내
- 스팸 방지: 제목+내용 최소 글자 수 검증
- app.py의 sidebar 스타일과 일관된 Community 섹션 버튼 추가

---

## 3. 범위 (Scope)

### In Scope
- `page10.py` 신규 생성 (아이디어 제안소 페이지)
- `app.py` 수정:
  - `IDEA = "아이디어 제안소"` 상수 추가
  - Community 섹션에 버튼 추가
  - `page_to_run_map`에 매핑 추가

### Out of Scope
- 사용자 인증/로그인 시스템
- 아이디어 추천(좋아요) 기능
- 이메일 알림
- 이미지 첨부

## 4. 데이터 모델 (Data Model)

**Collection**: `ideas` (MongoDB, `dlc` DB)

```json
{
  "_id": "ObjectId",
  "title": "string (optional, max 100자)",
  "content": "string (required, min 10자, max 500자)",
  "created_at": "datetime (UTC)",
  "status": "string (제안됨 | 검토중 | 채택 | 보류)"
}
```

## 5. UX 설계 (UX Design)

### 제안 폼

```
┌─────────────────────────────────────────────┐
│  💡 아이디어 제안소                           │
│  누구나 자유롭게 아이디어를 남겨주세요.         │
│                                             │
│  제목 (선택)                                 │
│  ┌─────────────────────────────────────┐    │
│  │ 예: AI 회의록 자동 요약 도구          │    │  ← placeholder
│  └─────────────────────────────────────┘    │
│                                             │
│  아이디어 내용 *                             │
│  ┌─────────────────────────────────────┐    │
│  │ 예: 회의 중 녹음을 AI가 실시간으로    │    │  ← placeholder
│  │ 요약해주는 기능이 있으면 좋겠습니다.   │    │
│  │ 카카오톡 공유도 되면 더 좋을 것 같아요│    │
│  └─────────────────────────────────────┘    │
│                                             │
│         [ 제안하기 ]                         │
└─────────────────────────────────────────────┘
```

### 관리자 뷰
- 아이디어 목록 (최신순)
- 상태 배지: `제안됨` / `검토중` / `채택` / `보류`
- 상태 변경 버튼 + 삭제 버튼
